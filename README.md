# SafarGov - Govt app

3 users:
- Elected representative
- Public Works Department
- Contractor

Features  include:
- Creating a workflow and bridging the gap of communication between all stakeholders
- Report generation of all potholes and sorting  according to priority and intensity of pothole
- Download and share report as PDF
- In app communication - Sending reports to all other stakeholders
- OTP login
- Real time map and navigation to detected pothole
- Confirmation of fixed potholes by authenticating images
